---
title: Configure App Card Packages
---

Configuring App card packages enables users to select and purchase card services directly in the App, flexibly matching different markets and pricing strategies.

Go to **Card Services** > **Card Management** > **App Card Packages**

Create one or more consumer-facing card packages based on your business needs. The settings will be displayed in the white-label App for users to select and apply.

<Warning>
Before creating an App card package, you must set the card refund coin.
If a refund transaction occurs, the system will return the equivalent amount of digital currency to the user's account based on the refund amount and the coin exchange rate.
</Warning>

## Configuration Details

**1. Basic Package Information (Chinese & English)**

* Package name
* Package description
* Package label (e.g., "Basic", "VIP Exclusive")

**2. Package Fees**

* Member card issuance fee (in USD)
* Various transaction fees

Package pricing can be set based on your operational strategy and combined with referral and rebate campaigns.
