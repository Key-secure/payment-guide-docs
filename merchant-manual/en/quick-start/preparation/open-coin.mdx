---
title: Add Supported Coins
---

Configure supported coins to enable deposit and withdrawal for merchant-available currencies, and set transaction rules to ensure secure fund operations and smooth refund processes.

![Add Supported Coins](/en/images/coin.png)

Go to **Operations Management** > **Merchant Coin Management**, add the coins you want to support, and configure the following parameters:

* **Enable Deposit / Withdrawal:** When enabled, end users can deposit or withdraw using this coin.
* **Minimum Deposit Amount:** Set the minimum deposit threshold; deposits must meet this amount to be credited.
* **Minimum Withdrawal Amount:** Set the minimum withdrawal threshold to limit end-user withdrawals.
* **Withdrawal Fee:** Fee deducted from users when they withdraw.
