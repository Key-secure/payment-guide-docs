title: KYC记录
---

## I. 功能简介

实时查看平台会员的身份认证记录及验证详情，追踪各类KYC服务的使用状态，确保平台合规运行。

## II. 页面说明

| 字段名称 | 描述 |
|------|------|
| 会员ID / 邮箱 | 显示会员唯一标识及注册信息 |
| 提交时间 | 用户发起KYC认证请求的时间 |
| 认证状态 | 当前认证进度状态（如：审核中、已通过等） |

<Note>
每次KYC服务调用（即用户提交验证）都会消耗认证额度，请合理安排调用频次。
</Note>

**查看KYC认证详情**
点击任意认证记录进入详情页，可查看如下字段：

| 字段名称 | 描述 |
|------|------|
| 会员ID/邮箱 | 对应会员的唯一标识 |
| SumsubID| Sumsub生成的唯一认证流水号 |
| 认证状态 | 与Sumsub接口实时同步的验证状态 |
| 用户姓名| 用户提交的验证的姓名信息 |
| 验证文档类型| 用户提交的验证身份类型 |
| 时间| 用户提交的验证的时间 |


<Note>
系统仅根据Sumsub SDK配置展示部分关键字段，如需查看完整认证详情，请登录Sumsub控制台或联系KeySecure支持团队。
</Note>
