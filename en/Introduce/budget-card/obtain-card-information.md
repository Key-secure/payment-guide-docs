## Overview

**Function Name**: get_card_info
**Description**: Retrieve core information of the payment card

## Return Data Details

### Required Information List
- Card Number
- Expiration Date
- CVV Security Code
- Cardholder Name

### Query Path
PayFun App → Card Management → View Card Details

### Security Reminder
Card number, expiration date, and CVV are sensitive data and must not be disclosed to third parties
