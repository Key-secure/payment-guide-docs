title: 会员列表
---

## I. 功能简介

会员列表模块用于帮助机构全面掌握平台会员（即最终用户）的基本身份信息、资产状态与卡片使用情况，并支持对个体会员账户执行权限管理与安全控制操作。该模块是日常风控与用户支持的重要入口。

通过此功能，机构可快速筛选会员、查阅关键账户信息，并灵活执行提现额度调整、冻结操作、解绑安全认证等管理行为，从而提高运维效率与客户服务能力。


## II. 页面说明

| 信息 | 说明 |
|------|------|
| 用户ID / 邮箱 | 显示会员唯一标识及注册信息 |
| 当前账户余额 | 展示用户账户内剩余可用数字货币的资产余额 |
| 卡片列表 | 展示用户名下的所有虚拟卡片状态 |
| 状态操作栏 | 包括账户功能控制按钮（如提现冻结、登录冻结等） |




## III. 支持操作说明

### 修改提现额度（24小时）


操作说明：输入新的24小时内最大可提现额度，系统将实时生效。

应用场景：根据风险等级或业务需求调整提现权限


### 更换注册邮箱

应用场景：用户请求修改邮箱、原邮箱不可用等情况

### 冻结登录权限


操作说明：确认冻结后，该账户将无法登录平台，可通过后台手动解除冻结。


应用场景：怀疑账号被盗、用户主动请求封停等。

### 冻结提现权限


操作说明：该操作将立即限制用户提出资金。

应用场景：用户身份存疑、触发风险模型等。

### 解绑谷歌验证（GA）


操作说明：操作需由管理员人工确认，并记录原因，用户后续需重新绑定GA后才能完成受限操作。

应用场景：用户遗失手机、无法通过GA验证等。
