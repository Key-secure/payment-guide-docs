## 功能概述

“充值记录”页面用于展示 会员通过链上方式向商户 MPC 钱包地址充值数字资产 的完整记录，帮助商户及时确认到账状态、核查充值路径，并作为资金入账的重要参考。

该模块结合链上监听与内部会员体系，可确保充值数据的 透明、实时、可追踪，是数字货币进账的关键入口。

### 充值记录字段

每笔充值记录将包含以下主要信息字段：

| 字段 | 描述 |
|------|------|
| 会员 ID | 发起提现请求的会员唯一编号 |
| 充值币种 | 用户充值的加密货币 |
| 链名称| 充值所使用的区块链网络，如 Tron、Ethereum、Polygon 等 |
| 充值数量 | 本次充值的资产数量，按用户实际转账数量记录 |
| 充值地址 | 系统分配给该会员的收款地址（MPC 钱包子地址）|
| TXID	 | 区块链上的交易编号，可点复制至浏览器查看详情|
| 状态| 确认中 / 成功 / 失败|



## 充值流程说明

**地址分配**

每位会员在系统中拥有专属的 MPC 钱包收款地址。

支持按币种 / 网络分配不同地址，确保充值路径清晰、可识别。

**发起充值**

会员在外部钱包发起链上转账，将数字货币发送至其专属充值地址。

**链上监听**

系统实时监听各链交易，捕捉到账交易并比对 TXID、充值地址及金额。

**入账确认**

若交易符合条件（如确认数达到设定门槛，充值数量超过最小充值数），系统自动识别并更新会员总资产。


<Warning>

链上充值为不可逆操作，请确保用户通过系统分配的正确地址充值，避免资产丢失。
</Warning>
