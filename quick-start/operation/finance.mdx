title: 管理财务记录
---

## I. 功能概述

商户可通过访问财务管理，查看充值记录、提现记录、卡片消费记录。此模块掌握资金流转动态、验证交易合法性、完成日常财务核对与异常排查。

## II. 适用角色

* 财务人员

* 商户管理员

* 风控/合规审核人员


## III. 记录简介

**1. 充值记录**

用于记录会员向商户 MPC 钱包地址发起的数字货币充值。

主要字段包括：

* 会员 ID

* 充值币种（如 USDT, ETH）

* 链名称（如 Tron, Ethereum）

* 充值数量

* 充值地址

* TXID（交易哈希）

* 状态

<Note>
仅当链上确认数达到设定阈值后，该充值才会被系统认定为成功；

成功充值的金额将自动计入用户账户余额；

若充值失败或未完成确认，将无法入账。

建议财务人员定期核对充值到账金额与链上实际交易记录，确保资金一致性。

</Note>

**2. 提现记录**

展示用户向链上地址发起的提现请求，该操作需后台人工审批。

主要字段包括：

* 会员 ID

* 提现币种 / 网络

* 提现金额

* 提现地址

* TXID（链上交易哈希）

* 审批状态（待审批 / 已通过 / 拒绝）

* 审批人 / 时间

<Note>

提现申请需由管理员手动审批（见提现审批操作流程）；

系统将在审批通过后，自动调用 MPC 钱包完成链上转账，并完成 Co-signer 签名；

请务必确保您的 MPC 钱包已启用归集功能，且加油站钱包中有充足矿工费以完成链上交易。

提现失败多因矿工费不足或归集未完成，请定期检查相关钱包状态。
</Note>

**3. 消费记录**

记录所有通过虚拟支付卡完成的交易，包括授权、结算、退款等阶段。

主要字段包括：

* 交易类型

* 交易时间与状态

* 发起会员信息（ID、昵称）

* 卡片信息（卡号 / 类型）

* 消费金额与币种

* 商户名（消费发生地）

涉及账户变动明细：1. 商户预算账户余额变动 2.会员总资产变动
